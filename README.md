## 功能介绍

- 🤖 **实时作战辅助 (Real-time Task)**：全自动开荒，解放双手。
    - **自动剧情**：智能跳过、分支选择、加速播放，沉浸式（或快速）体验。
    - **交互处理**：自动点击指引光标（老普之手）、翻阅说明书、点击继续、关闭档案语音。
    - **辅助战斗**：自动普攻、技能、终结技、连携技，支持锁定目标。
    - **自动拾取**：自动拾取掉落物与交互采集。
    - **便捷操作**：快速传送确认、自动长距滑索。
- 🧩 **拼图大师 (Puzzle Solver)**：高精度识别并自动完成各类场景谜题，支持单次、重复和演示模式。
- 💰 **自动贸易 (Auto Resell)**：智能识别物资利润，自动执行低买高卖策略，跨地区贸易从未如此简单。
- 📐 **蓝图搬运工 (Blueprints)**：批量导入建筑蓝图，支持智能解析分享码，快速构建工业流水线。
- 📅 **日常自动化 (Daily Automation)**：一键领取每日奖励、自动阅读百科红点、领取生产手册奖励、完成 Baker 入口任务。
- 🎮 **多端适配 (Cross Platform)**：完美支持 PC 端 (Win32 API) 与安卓端 (ADB)，支持自动启动并进入游戏。
